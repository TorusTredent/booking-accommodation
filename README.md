# Booking-accommodation

A web application that allows you to find a hotel and book a room (s) in it. Hotels are selected according to the selected filters, as well as using the selected period of stay. 

Prerequisites:
* JDK 1.8+
* Maven

Technology stack:
* Spring Boot;
* Spring Web;
* Spring Data JPA;
* H2;
* Logging SLF4J;
* Mapping;
* Bean Validation;
